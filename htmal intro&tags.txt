[intro html ]

 HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web  pages. HTML describes the structure of a Web page. HTML consists of a series of elements. HTML  elements tell the browser how to display the content.

What is HTML basic tag in use?
 <html>: This is the root element of an HTML document, which contains all the other elements.
 <head>: This element contains metadata about the document, such as the title, charset,and links to  external stylesheets or scripts.
 <body>: This element contains the content of the web page, such as text, images, and other elements.
 <h1> to <h6>: These elements are used to define headings and subheadings on the web page.
 <p>: This element is used to define a paragraph of text.
 <ul> and <ol>: These elements are used to create unordered and ordered lists, respectively.
 <li>: This element is used to define a list item within a list.
 <a>: This element is used to create hyperlinks to other web pages or resources.
 <img>: This element is used to insert images into the web page.
 <table>: This element is used to create tables with rows and columns of data.

[HTML Block and Inline Elements]

HTML Elements are classified as Block Level and Inline Level elements on the basics of their display. Some elements display as blocks and some inline.

HTML Block level elements are elements who behave like blocks, like <p>, <h1>, <div>, <ul>, <ol>, <pre> and <address>. These elements always starts from a new line and occupy full width of parent element. 

<p> <div> <h1> < h2> <address> etc are block level elements,
<b>, <strong>, <i>, <span>, <u> and <s> are inline level elements.

What is blockquote tag in HTMl?
 The <blockquote> element defines a block of text that is a direct quotation. The <quote> element  should be used when a quotation is presented inline with the surrounding text, but when the quotation  is presented as a separate paragraph.
